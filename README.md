# AuditWisen AI

AuditWisen AI is a Next.js frontend for audit upload and AI-assisted audit workflows.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
